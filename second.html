1. What is DOM and what is the purpose of DOM?

DOM is a Document Object Model. programming interface for web documents. It page programs can change the document's structure, style, and content. DOM represents the document as nodes and objects

2. How will you select HTML elements using DOM? Name the DOM methods?

I am 5 ways select elements in DOM using selectors. 1. getElementsByTagName() 2. getElementsByClassName() 3. getElementById() 4. querySelector() 5. querySelectorAll()

3. What is event bubble?

Bubbling is a concept in the Document Object Model that event bubbles up to its parent and ancestor elements in the DOM tree until it gets to the root element.

4. What is a callback function and why will you use it?

Callback function is a function passed into another function as an argument.
